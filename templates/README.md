# Booking Excel Template

booking-template.xlsx is the source-of-truth Excel booking layout.

Future Excel export must preserve:
- sheet names
- Arabic labels
- merged cells
- colors
- borders
- column widths
- RTL layout
- date formats
- Hijri/Gregorian layout

The main booking sheet appears to be:
ورقة1

The layout uses:
- chalet blocks side by side
- 9 columns per chalet block
- two rows per date
- period / phone / amount triples

Do not replace this template with a generic workbook layout unless explicitly approved.
