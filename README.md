# نظام حجوزات الشاليهات - النسخة النهائية

هذا المشروع تطبيق عربي RTL ثابت يعمل على GitHub Pages و iPhone Safari لإدارة حجوزات الشاليهات.

## الرابط النهائي الوحيد

استخدم هذا الرابط فقط:

```text
https://qw1qw66-sudo.github.io/index.html/sync-cloud/final.html
```

`app.html` يحوّل تلقائيًا إلى هذا الرابط.

## طريقة المزامنة

النظام لا يستخدم البريد الإلكتروني ولا Magic Link ولا SMTP.

المزامنة تتم فقط عبر:

```text
Workspace Code + PIN
```

ويتم الاتصال بـ Supabase عبر RPC فقط:

```text
get_shared_workspace(p_workspace_key, p_access_pin)
save_shared_workspace(p_workspace_key, p_access_pin, p_data)
```

## إعداد Supabase

افتح:

```text
Supabase → SQL Editor → New query
```

ثم شغّل الملف:

```text
database/shared_workspace_sync.sql
```

هذا ينشئ جدول:

```text
shared_workspaces
```

ويضيف دوال السحب والرفع.

## طريقة الاستخدام

### أول جهاز

1. افتح الرابط النهائي.
2. أدخل رمز المساحة مثل `ALI6`.
3. أدخل PIN من 4 خانات أو أكثر.
4. إذا كانت المساحة موجودة سيتم تحميل بياناتها.
5. إذا لم تكن موجودة، لن تظهر بيانات ولن يحدث رفع تلقائي.

### بعد تحميل مساحة صحيحة

- يمكنك إضافة شاليهات.
- يمكنك إضافة فترات داخل الشاليه.
- يمكنك إضافة حجوزات.
- الحفظ محلي فقط حتى تضغط: `رفع التعديلات`.

### جهاز ثاني

1. افتح نفس الرابط.
2. أدخل نفس Workspace Code ونفس PIN.
3. سيتم تحميل نفس بيانات المساحة.

## قواعد حماية البيانات

- لا توجد بيانات افتراضية.
- لا يتم إنشاء شاليهات الواحة أو الياسمين.
- لا يتم رفع أي شيء عند فتح التطبيق.
- لا يتم الرفع قبل تحميل مساحة صحيحة.
- يتم منع رفع نسخة فارغة فوق بيانات سحابية موجودة.
- يتم حفظ Backup محلي قبل أي رفع للسحابة.
- استيراد JSON لا يرفع تلقائيًا.

## هيكل البيانات

البيانات داخل Supabase تحفظ كـ JSON:

```json
{
  "schema_version": 3,
  "settings": {},
  "chalets": [],
  "bookings": []
}
```

كل شاليه يحتوي بياناته الخاصة:

- رقم التواصل
- اسم العامل
- رقم العامل
- رابط Google Maps
- الشروط
- الفترات والأسعار

## الفترات والأسعار

الفترات داخل كل شاليه. كل فترة لها:

- label
- start
- end
- weekday_price
- weekend_price

الويكند هو: الخميس والجمعة والسبت.

## السند

السند يعرض بيانات الشاليه المرتبط بالحجز فقط. لا يعرض رقم الحجز ولا الملاحظات الداخلية.
