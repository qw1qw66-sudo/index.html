التقارير الشهرية

ملف Excel الأونلاين

يفتح أحدث ملف حجوزات Excel المُولَّد على الخادم للسنة الحالية (للقراءة والتنزيل فقط).

لماذا لا يظهر الحجز في Excel؟
  • ملف Excel يعتمد على البيانات المرفوعة للسحابة، وليس التغييرات المحلية غير المرفوعة.
  • يجب الضغط على رفع التعديلات قبل توقع ظهور الحجز في Excel.
  • التحديث الأونلاين عبر GitHub Action قد يحتاج بعض الوقت.
  • الحجوزات المؤكدة فقط تظهر في Excel.
  • الحجوزات الملغية أو المحذوفة أو المعلقة لا تظهر في Excel.
  • القالب الحالي يدعم 4 فترات فقط لكل شاليه في اليوم.
  • الفترة الخامسة والسادسة لا تظهر في Excel حاليًا.
  • تقرير التصدير يوضح المكتوب والمتجاوز والمتعارض.
  • الرابط الحالي يفتح ملف السنة الحالية: exports/bookings-YYYY.xlsx

تفتح صفحة Supabase الرسمية. اختر مشروع الشاليه، الصق أسماء المفاتيح وقيمها، ثم اضغط Save. لا تُحفظ المفاتيح داخل برنامج الشاليه.

بعد إضافة المفاتيح، افتح GitHub Actions وشغّل Deploy Supabase Staging.

١) مشروع Staging (بيئة تجريبية منفصلة)

  1. سجّل الدخول إلى Supabase.
  2. أنشئ مشروعاً جديداً باسم قريب من: chalet-booking-staging
  3. اختر منطقة قريبة من السعودية.
  4. احفظ كلمة مرور قاعدة البيانات في مدير كلمات المرور.
  5. بعد إنشاء المشروع ارجع هنا واضغط: تم إنشاء Staging

انسخ من صفحة إعدادات المشروع القيمتين غير السرّيتين فقط: معرّف المشروع (Project Ref) والمفتاح العام (publishable). لا تنسخ كلمة مرور قاعدة البيانات ولا أي مفتاح سرّي إلى هنا.

٢) أسرار النشر في GitHub

السرّان الموجودان مسبقاً يُعاد استخدامهما كما هما: SUPABASE_URL و SUPABASE_ANON_KEY. أضف فقط السرّين الناقصين:
SUPABASE_ACCESS_TOKEN — أنشئه من صفحة Access Tokens في حساب Supabase.
SUPABASE_DB_PASSWORD — كلمة مرور قاعدة البيانات للمشروع.
لا تضف أبداً مفتاح SERVICE ROLE — الخادم يستلمه من Supabase تلقائياً ولا يدخل GitHub أو المتصفح.

خطوات الإعداد من الجوال

  1. أنشئ مشروع Staging ثم اضغط «تم إنشاء Staging».
  2. الصق معرّف المشروع والمفتاح العام واحفظ.
  3. افتح صفحة إضافة المفاتيح، اختر مشروع Staging، الصق القائمة مع مفتاح DeepSeek واضغط Save.
  4. افتح أسرار GitHub للنشر وأضف الأسرار الأربعة.
  5. افتح صفحة النشر وشغّل Deploy Supabase Staging.
  6. افتح نسخة Staging، سجّل حساباً تجريبياً، واضغط فحص الربط.
  7. جرّب سؤال: ما هي حجوزات اليوم؟
